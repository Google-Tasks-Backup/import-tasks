Background information about Google Tasks data

Note that this information is based on my own discoveries and conclusions as at 27 Feb 2012. I hope that this information helps people, but I cannot guarantee the accuracy or completeness of this information. Note also that Google may change their specifications, in which case this information may be out of date.

Understanding the "Raw" data format

Table 1 lists the various states and the corresponding task attribute values. It is possible to select to download only tasks which match specified task attributes when initiating a backup with {{ app_title }}
Normally, when downloading for applications such as Outlook or iCalendar, uncheck deleted and hidden (note 1). Check completed if you wish to include completed tasks (i.e., to keep a backup of your completed tasks).

Notes
Note 1 - The Outlook and iCalendar file formats do not have fields for deleted or hidden, so programs may display tasks which are not normally visible.
Note 2 - When tasks are deleted, they are marked as Hidden and moved to the [Trash] view.
According to Google, deleted tasks stay in the Trash (deleted = True) for "some period of time" and are then auto deleted. I have noticed deleted task up to six months old. There does not appear to be any way to force a task to be actually deleted. That is, the Trash bin cannot be emptied by the user.

To summarise, a task can be in any one of the following states; ** needsAction (unchecked) ---- Visible in main list ** completed (checked) {before clicking [Clear completed]} ---- Visible in main list ** completed (checked) {after clicking [Clear completed]} ---- Visible only in [Completed tasks] ** Deleted ---- Visible only in [Trash] -> Marked complete, and then clicked Clear completed = completed, hidden -> Marked complete and deleted (appears in [Trash]) = completed, deleted - Investigate calling ReturnResultsHandler.post() from ReturnResultsHandler.get() -- Sometimes the get() handler is being called
Note 1 - When checked, the backup includes completed tasks. If unchecked, only incomplete tasks are exported.
Note 2 - When tasks are deleted, they are marked as Hidden and moved to the [Trash] view.
Outlook and iCal do not differentiate deleted tasks, so they will appear as normal tasks.
Note 3 - According to Google, deleted tasks stay in the Trash (status = deleted) for "some period of time" and are then auto deleted. I have noticed deleted task up to six months old. There does not appear to be any way to force a task to be actually deleted. That is, the Trash bin cannot be emptied by the user.
Note 4 - When tasks which have been marked complete, and then [Clear completed] is clicked, those tasks are marked as Completed AND Hidden.
Outlook and iCal do not differentiate hidden tasks, so they will appear as normal tasks.
