The service

Julie Smith ("me", "I") operates the Google Tasks Import ("GTI") and Google Tasks Backup ("GTB") websites at https://import-tasks.appspot.com and https://tasks-backup.appspot.com

  • GTI allows a user to import tasks from a correctly formatted user-uploaded file to the Google Tasks server.
    • GTI suppports the GTB "Import/Export" GTBack and CSV file formats.
  • GTB allows a user to retrieve their tasks from the Google Tasks server, and view or export their tasks in a number of different formats.
    • The "Import/Export" GTBack and CSV formats are specifically designed to be used by GTI.

These services are provided free of charge, as they are running as free Google App Engine applications. Google have generously extended the free quota for GTI and GTB, so that other people can use GTI and GTB.

This privacy policy covers GTI. The privacy policy for GTB is at https://tasks-backup.appspot.com/privacy

Collection and usage of data

In order to provide this service, the user is authenticated with Google, which provides GTI with the following;
  • Temporary access to be able to add and delete tasks for the user
    • This is required in order to be able to import tasks
    • Access to a user's tasks automatically expires, as specified during the Google authentication process
  • The user's name and email address
    • This is the name and email address associated with the Google account that is used when the user authenticates with Google
    • GTI uses the email address to send an email when the import process is complete
    • The account name and email address is stored in logs, which may be used by the system administrator to investigate user-reported problems
To use GTI, the user uploads a file containing tasks to be imported into the user's Google Tasks.
  • The uploaded file is temporarily stored by GTI, and then data for each task is read from the uploaded file and a new task is then created for each row of task data.
  • When all tasks have been created, the temporary file is deleted.

Data security

GTI uses standard HTTPS for all communications between the browser and the GTI application.

Server logs

Google App Engine automatically logs information about each request to GTI. This information includes;
  • Time of the request
  • IP address
  • User name
  • Browser version
  • Operating system

If task creation fails, the details of the task that failed may be stored in the server log, along with the user's email address, to enable debugging and application improvements.

Only system administrators have access to server logs. The user's task details and email address will not be shared with anyone.

Cookies

Cookies are used for two purposes in GTI;
  • Google uses cookies to track your authorisation credentials between pages. This ensure that you (and only you) can access your Google account details.
  • Google Analytics is used to track usage such as; number of unique visitors, location, browser type, operating system.
